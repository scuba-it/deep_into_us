# deep_into_us
 * scuba-it은 scuba diving과 관련한 정보를 공유합니다.
 * 편집에 자유롭게 참여 가능 합니다. 
 * 단순한 정보의 갱신은 "Pull requests" 기능을 사용할 수 있습니다.
 * 자주 업데이트 하고 싶다면 권한을 요청하세요.
